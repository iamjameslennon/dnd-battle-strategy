# ⚔️ D&D Battle Strategies

Battle strategy references for each character in our D&D campaigns. Each character has a visual SVG flowchart to guide decision-making in combat and an accompanying PDF for easy printing.


## 📂 Structure

```
├── characters.md              # Full party stats & D&D Beyond links
├── STYLE-GUIDE.md             # Visual style guide for strategy sheets
├── _template.svg              # Base template for new strategy sheets
├── Ael Varris/
│   ├── ael-varris-battle-strategy.svg
│   └── ael-varris-battle-strategy.pdf
├── Claude Skamos/
│   ├── claude-skamos-battle-strategy.svg
│   └── claude-skamos-battle-strategy.pdf
├── Clover Darkbloom/

│   └── clover-darkbloom-battle-strategy.pdf
├── Idhren/
│   ├── idhren-battle-strategy.svg
│   └── idhren-battle-strategy.pdf
├── Lysander/
│   ├── lysander-battle-strategy.svg
│   └── lysander-battle-strategy.pdf
└── Oskar/
    ├── oskar-battle-strategy.svg
    └── oskar-battle-strategy.pdf
```

Each SVG is a combat flowchart tailored to that character's abilities, spells, and role in the party. PDFs are also available for each character for easy printing and reference during sessions.
