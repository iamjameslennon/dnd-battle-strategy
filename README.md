# ⚔️ D&D Battle Strategies

Battle strategy references for each character in our D&D campaigns. Each character has a visual SVG flowchart to guide decision-making in combat and an accompanying PDF for easy printing.


## 📂 Structure

```
├── characters.md              # Full party stats & D&D Beyond links
├── STYLE-GUIDE.md             # Visual style guide for strategy sheets
├── _template.svg              # Base template for new strategy sheets
├── Ael Varris/
│   ├── Ael Varris battle strategy (Page 1).svg
│   └── Ael Varris battle strategy.pdf
├── Claude Skamos/
│   ├── Claude Skamos battle strategy (Page 1).svg
│   └── Claude Skamos battle strategy.pdf
├── Clover Darkbloom/
│   ├── Clover Darkbloom battle strategy (Page 1).svg   # main flow
│   ├── Clover Darkbloom battle strategy (Page 2).svg   # Wild Shape forms (back)
│   └── Clover Darkbloom battle strategy.pdf            # 2-page, print double-sided
├── Idhren/
│   ├── Idhren battle strategy (Page 1).svg
│   └── Idhren battle strategy.pdf
├── Lysander/
│   ├── Lysander battle strategy (Page 1).svg
│   └── Lysander battle strategy.pdf
└── Oskar/
    ├── Oskar battle strategy (Page 1).svg
    └── Oskar battle strategy.pdf
```

**Naming convention.** Each page is an SVG named `<Character> battle strategy (Page N).svg`
(most sheets are a single `(Page 1)`; multi-page sheets like Clover add `(Page 2)` etc.).
The printable PDF for a character is `<Character> battle strategy.pdf` — for multi-page
sheets it contains every page in order, ready to print double-sided.

Each SVG is a combat flowchart tailored to that character's abilities, spells, and role in the party. PDFs are also available for each character for easy printing and reference during sessions.
